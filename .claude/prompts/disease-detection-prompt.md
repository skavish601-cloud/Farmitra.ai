# Prompt Template: Disease Detection

```
Analyze the uploaded crop photo for signs of pest damage or disease.

- Identify the likely pest/disease, with a confidence level.
- If confidence is low, say so explicitly rather than guessing a
  specific diagnosis.
- Suggest both organic and chemical treatment options where applicable,
  clearly labeled.
- Recommend consulting a local agricultural officer before applying any
  chemical treatment, given the stakes of an incorrect dosage.
- Respond in the farmer's selected language (Hindi/English).
```

Pairs with the AI Engineer agent's confidence-threshold guardrail requirement.
